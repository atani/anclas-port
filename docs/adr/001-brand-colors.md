# ADR-001: クラブカラーの採取と確定

## ステータス

承認済み（2026-06-28）

## コンテキスト

アプリの UI カラーをアンクラスのクラブカラーに揃える必要がある。design.md では「公式サイト・ユニフォームから採取」「申請前に確定」と方針だけ記載し、具体値は未定だった。

## 決定

エンブレム画像（512×512 PNG）のカラーヒストグラム分析と anclas.jp の CSS から以下の3色を採取し、アプリのカラーパレットとして確定した。

| 名前 | HEX | RGB | 出典 | 用途 |
|---|---|---|---|---|
| ダークネイビー | `#162D66` | (22, 45, 102) | エンブレム内側（錨・文字） | ナビバー・カード見出し・主要テキスト |
| ブルー | `#1970C1` | (25, 112, 193) | エンブレム外輪 | 自チームハイライト・アクセント |
| ゴールド | `#DD9933` | (221, 153, 51) | anclas.jp アクセント | 補助アクセント（将来用） |
| 勝 | 緑（system green） | — | 一般的なサッカーアプリ慣習 | WIN 表示 |
| 分 | 灰（system gray） | — | 同上 | DRAW 表示 |
| 負 | 赤（system red） | — | 同上 | LOSE 表示 |

### 採取方法

1. エンブレム PNG を ImageMagick で 200×200 にリサイズし、8色に減色してヒストグラムを取得
2. 上位2色（白を除く）をダークネイビー `#162D66`（4471px）とブルー `#1970C1`（4300px）として採取
3. anclas.jp トップページの CSS から `background-color:#273349`（ヘッダー）と `background-color:#dd9933`（アクセント）を確認
4. `#273349` はエンブレムの `#162D66` より明るく灰味が強い中間色だったため、エンブレム基準の `#162D66` を正として採用
5. ゴールド `#DD9933` は anclas.jp 固有のアクセント色として保持

### 不採用とした候補

- `#273349`（anclas.jp ヘッダー背景）: WordPress テーマの汎用色に近く、エンブレムのコア色と乖離していたため不採用。ただし近似色であり、将来の微調整で戻す可能性はある

## 影響

- `ios/AnclasPort/Theme.swift` の `navy` / `blue` / `gold` 定数
- `ios/AnclasPort/Assets.xcassets/AccentColor.colorset/` の sRGB 値
- 以降の全画面（日程・順位・選手タブ）はこのパレットに従う
